# 🏥 Hospital Management SQL Project

## 🚀 Overview

This project demonstrates a complete SQL learning journey from **basic queries to advanced analytics**, using a Hospital Management dataset.

It includes real-world business problems such as patient analysis, revenue tracking, appointment trends, and advanced SQL techniques.

---

## 📊 Key Features

### 🔹 Basic SQL

* SELECT, WHERE, ORDER BY
* GROUP BY, HAVING
* Aggregate functions (SUM, COUNT, AVG)

### 🔹 Joins

* INNER JOIN
* LEFT JOIN
* SELF JOIN

### 🔹 Intermediate Concepts

* Subqueries
* DISTINCT
* CASE WHEN
* Date & String Functions

### 🔹 Advanced SQL

* Window Functions

  * ROW_NUMBER()
  * RANK()
  * DENSE_RANK()
  * LAG()
* CTEs (WITH clause)
* Running totals
* Gap analysis
* Consecutive visits (streak detection)

### 🔹 Set Operations

* UNION
* UNION ALL
* INTERSECT
* EXCEPT

### 🔹 Data Manipulation

* INSERT
* UPDATE
* DELETE

### 🔹 Schema Modifications

* ALTER TABLE
* ADD / MODIFY / DROP columns

---

## 📈 Business Problems Solved

* 🔍 Patients with consecutive visits
* 📅 Appointment gap analysis
* 💰 Revenue by doctor and city
* 🏆 Top-performing doctors
* 📊 Monthly revenue trends
* 🧑‍⚕️ Patient lifetime value (LTV)
* ⚠️ High-risk patients (high visits + high spending)
* 📉 No-show rate calculation
* 📍 Cohort analysis

---

## 🧠 Key Learnings

* Writing efficient SQL queries for real-world scenarios
* Using window functions for advanced analytics
* Structuring queries using CTEs
* Handling duplicates and edge cases
* Translating business problems into SQL logic
